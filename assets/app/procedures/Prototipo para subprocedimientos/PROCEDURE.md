# Prototipo para subprocedimientos

Este procedimiento está pensado para demostrar el uso de subprocedimientos.

## Idea

Prototipo pensado para:

- abrir un diálogo
- iniciar varios subprocedimientos
   - con la posibilidad de cerrar el procedimiento padre
   - y demostrar que los subprocedimientos se cierran también

## Dependencias

Se inyectan los componentes:

- `Vue.options.components.ProcedurePrototipoParaTesterYProgressBarViewer`
- `Vue.options.components.ProcedurePrototipoParaTesterYProgressBarForm`
