# Documentación de la aplicación

Esta es la documentación de la aplicación.

## Comandos de aplicación

La API de `aplicacion-base`, por su banda, inyecta estos eventos de teclado:

- `ALT + J`: abre un diálogo con el buscador de procedimientos (`App Procedures API`)

La API de `NwtFramework`, por su banda, inyecta estos eventos de teclado:

- `CTRL + SUPR`: abre un diálogo con el gestor de procesos (`Nwt Process Manager API`)
- `ALT + L`: abre un diálogo con las configuraciones globales (`Nwt Settings API`)
