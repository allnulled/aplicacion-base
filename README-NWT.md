# Documentación de NWT

NWT son las siglas de *Node Webkit Toolkit*.

NWT es un proyecto de código en JavaScript para navegador que sirve de base para aplicaciones basadas en:

- navegador normal como `firefox` o `google-chrome` para aplicaciones web
- navegador con `nw.js` para aplicaciones de escritorio
- navegador con `cordova` para aplicaciones de móvil

# Librerías externas

El framework de NWT **incluye** las siguientes librerías externas:

- `vue@2`: para definir y renderizar componentes de la interfaz.
- `win7.css`: para estilos de interfaz.
- `marked.js`: para soportar *markdown*.
- `socket.io-client`: para el refresco automático en el modo desarrollo.
- `papaparse`: para la gestión de formatos CSV tanto en front-end como en back-end.
